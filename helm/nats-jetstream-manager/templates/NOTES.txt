Get the application URLs by running these commands:

  # Frontend
  kubectl port-forward svc/{{ include "nats-jetstream-manager.fullname" . }}-frontend {{ .Values.frontend.service.port }}:{{ .Values.frontend.service.port }} -n {{ .Release.Namespace }}

  # Backend API
  kubectl port-forward svc/{{ include "nats-jetstream-manager.fullname" . }}-backend {{ .Values.backend.service.port }}:{{ .Values.backend.service.port }} -n {{ .Release.Namespace }}

{{- end }}

NOTE: The frontend's NEXT_PUBLIC_API_URL is a build-time variable. If you need to
change the backend URL the frontend points to, you must rebuild the frontend image
with the updated NEXT_PUBLIC_API_URL environment variable.
